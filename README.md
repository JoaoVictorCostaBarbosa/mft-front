# MyFitTracker (MFT) – Frontend

O MyFitTracker (MFT) é uma aplicação web para acompanhamento de treinos e progresso físico.
Este repositório contém o frontend da aplicação, desenvolvido com Next.js, que consome a API do projeto.

## Tecnologias utilizadas

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Docker (em preparação)
- Integração com API própria

## Objetivo do projeto

Criar uma aplicação moderna para:

- Gerenciar treinos
- Registrar exercícios e séries
- Acompanhar evolução corporal

## Status do projeto

Em desenvolvimento

Funcionalidades e layout estão sendo implementados gradualmente.

## Como rodar o projeto localmente

1. Clone o repositório:

git clone https://github.com/seu-usuario/mft-frontend.git

2. Entre na pasta do projeto:

cd mft-frontend

3. Instale as dependências:

npm install
# ou
pnpm install

4. Rode o servidor de desenvolvimento:

npm run dev

5. Acesse no navegador:

http://localhost:3000
